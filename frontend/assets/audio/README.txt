To add a ringtone for the fake call feature:

1. Rename your ringtone file to 'ringtone.mp3'
2. Place it in this folder (assets/audio/)

You can get free ringtones from:
- https://notificationsounds.com/ringtones
- https://www.zedge.net/ringtones
- https://www.soundboard.com/sb/iPhone_Ringtones

Requirements:
- File format: MP3
- File name: ringtone.mp3
- Duration: Recommended 10-30 seconds
- File size: Keep under 1MB for better app performance

Note: Make sure you have the rights to use the ringtone in your application.
